@ Media Queries

Css uses the @media rule to detect information about the device being used to acess the website. For example, it can be used to detect the seive of the device, its capabilities, and its orientation. For this course we will be using it to detect the device size, but keep in mind that it is capable of much more.

phone: Width less than or equal to 480px

Tablet: Width beteen 481 px and 1023px

Desktop: Width greater than or equal to 1024px
